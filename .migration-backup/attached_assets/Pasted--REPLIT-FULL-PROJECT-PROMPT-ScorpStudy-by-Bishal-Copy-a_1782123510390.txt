## 🔑 API KEYS & ENVIRONMENT VARIABLES

The app uses these 3 keys. Store all of them in a `.env` file in the root of the project AND in Replit's **Secrets** tab (🔒 lock icon on sidebar):

```
SUPABASE_URL=https://xxxxxxxxxxxxxxxxxxx.supabase.co
SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### How to get each key:

**1. SUPABASE_URL & SUPABASE_ANON_KEY**
- Go to 👉 https://supabase.com and log in
- Open your project → click **"Project Settings"** (gear icon on sidebar)
- Click **"API"** under Project Settings
- Copy **"Project URL"** → this is your `SUPABASE_URL`
- Copy **"anon public"** key under Project API Keys → this is your `SUPABASE_ANON_KEY`

**2. GROQ_API_KEY**
- Go to 👉 https://console.groq.com and sign up (free, no credit card)
- Click **"API Keys"** on the left sidebar
- Click **"Create API Key"** → give it any name → copy it
- This is your `GROQ_API_KEY`

### How to add keys in Replit:
- Click the 🔒 **Secrets** icon on the left sidebar in Replit
- Add each key one by one:
  - Key: `SUPABASE_URL` → Value: your Supabase project URL
  - Key: `SUPABASE_ANON_KEY` → Value: your Supabase anon key
  - Key: `GROQ_API_KEY` → Value: your Groq API key
- In `app.py`, load them like this:
  ```python
  from dotenv import load_dotenv
  import os
  load_dotenv()
  
  SUPABASE_URL = os.environ.get("SUPABASE_URL")
  SUPABASE_ANON_KEY = os.environ.get("SUPABASE_ANON_KEY")
  GROQ_API_KEY = os.environ.get("GROQ_API_KEY")
  ```

### How to add keys for Vercel deployment:
- In your Vercel project → go to **Settings → Environment Variables**
- Add all 3 keys there with the same names
- Redeploy after adding them

## 🗄️ SUPABASE DATABASE TABLES

Create these tables in Supabase (add SQL comments in code):

```sql
-- Users handled by Supabase Auth

CREATE TABLE notes (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id uuid REFERENCES auth.users,
  title text,
  content text,
  created_at timestamp DEFAULT now()
);

CREATE TABLE chat_history (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id uuid REFERENCES auth.users,
  messages jsonb,
  created_at timestamp DEFAULT now()
);

CREATE TABLE quiz_results (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id uuid REFERENCES auth.users,
  topic text,
  score int,
  total int,
  questions jsonb,
  created_at timestamp DEFAULT now()
);

CREATE TABLE flashcards (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id uuid REFERENCES auth.users,
  topic text,
  cards jsonb,
  created_at timestamp DEFAULT now()
);

CREATE TABLE summaries (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id uuid REFERENCES auth.users,
  original_text text,
  summary text,
  key_points jsonb,
  created_at timestamp DEFAULT now()
);

CREATE TABLE generated_images (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  user_id uuid REFERENCES auth.users,
  prompt text,
  image_url text,
  created_at timestamp DEFAULT now()
);
```
